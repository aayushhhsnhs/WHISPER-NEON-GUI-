# 🎬 Subtitles Generator (Whisper GUI)

Modern local subtitles generator with a neon web UI powered by OpenAI Whisper.  
Runs entirely on your machine — no cloud, no API keys.

![UI Preview](preview.png)

---

## ✨ Features

- 🎧 Audio & video transcription
- 🌍 25+ languages supported
- ⚡ Apple Silicon GPU (MPS) acceleration
- 📄 Automatic SRT subtitle export
- 🎚 Model selection (tiny → large)
- 🎵 Waveform preview
- 📊 Progress indicator
- 💻 Fully local processing
- 🎨 Modern neon UI

---

## 🧠 Powered By

- OpenAI Whisper
- Flask
- Python
- FFmpeg
